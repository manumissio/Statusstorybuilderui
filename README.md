
  # Status Story Builder UI

  This is a code bundle for Status Story Builder UI. The original project is available at https://www.figma.com/design/XlprIW0HouZmetdt6OCVZR/Status-Story-Builder-UI.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Deploying to GitHub Pages

  This repo is configured to deploy from GitHub Actions when `main` is updated.

  1. In GitHub, open `Settings` -> `Pages`.
  2. Under `Build and deployment`, set `Source` to `GitHub Actions`.
  3. Push to `main` (or run the `Deploy to GitHub Pages` workflow manually).

  The workflow builds the app with a repo-aware base path and publishes `dist` to Pages.
